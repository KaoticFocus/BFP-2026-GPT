# BuildFlow Pro AI - Cursor Prompt Pack

This directory contains the complete prompt pack for building BuildFlow Pro AI.

## Usage

1. Start with `00-global-system-prompt.txt` - paste into Cursor rules
2. Run `01-bootstrap.txt` to scaffold the monorepo
3. Implement PRDs 02-21 in order
4. Run `23-e2e-demo.txt` to create the demo flow
